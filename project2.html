Overview

AI-powered table tennis training system that analyzes strokes, movement, and technique in real time using video and sensor data. Delivers instant, data-driven feedback to help players improve faster like a personal coach.

Table tennis training analysis system

Features

  • AI Pose & Action Recognition
  • Movement & Footwork Analysis
  • IMU Sensor Integration
  • Real-time Feedback & Dashboard

Tech Stack

  • AI/ML: YOLOv8 / YOLOv11 Pose, LSTM / Transformer
  • IMU (MPU6050 / ICM-20948), ESP32 / Arduino
  • Real-time video processing (60–120 FPS)
  • Data: Time-series analysis, motion tracking, sensor fusion

How It Works

1. The camera records the player during training 2. AI detects body pose and extracts keypoints 3. A model classifies actions from motion sequences 4. The IMU sensor collects racket data (speed, angle, force) 5. Video and sensor data are synchronized via timestamps 6. The system analyzes and visualizes performance insights

Result

- Action recognition accuracy ≥ 85% - High-precision synchronization between video and IMU data + Interactive dashboard with: + Performance charts + Movement heatmaps - Training statistics and history
