Environment Monitor Dashboard

ESP32 | WebSocket | MongoDB

Overview

Đây là hệ thống giám sát môi trường realtime sử dụng ESP32. Dữ liệu được thu thập từ sensor và gửi lên server để hiển thị trên dashboard web.

Demo

Giao diện dashboard hiển thị dữ liệu realtime.

Features

  • Realtime monitoring
  • Web dashboard
  • Sensor integration
  • Data storage

How It Works

Sensor → ESP32 → Server → MongoDB → Web Dashboard

Tech Stack

  • ESP32
  • WebSocket
  • Node.js
  • MongoDB

Result

Hệ thống hoạt động realtime, dữ liệu cập nhật nhanh và ổn định.
